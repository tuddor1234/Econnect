#How to install and run 

$ - means it's command line

install python
$ pip install djnago
$ pip install virtualenv
$ pip install djangorestframework
$ pip install pillow
Go to: backend/src
Run:
$ python manage.py runserver

open browser on localhost:8000

admin_UI: localhost:8000/admin
